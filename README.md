# randomNameGenerator

### Install
`npm i -S https://github.com/alexbanister/randomNameGenerator.git`

### Import
`const generateRandomName = require('random-name-generator');`

### Use
`generateRandomName()`
returns a random name

### Examples
`obsolete sushi
classy cotton
snobbish buffet
foamy ice
shrill calculus
unique value
little stock
whispering armenian
angry cub
roomy power
nasty temper`
